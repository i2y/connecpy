🔌 Connecpy

Python implementation of Connect Protocol for building type-safe RPC services

GitHub PyPI Documentation Connect Protocol

🚀 Quick Install

Install the protoc plugin with a single command:

curl -sSL https://i2y.github.io/connecpy/install.sh | bash

Custom Installation

Install to a custom directory:

# Install to user directory
curl -sSL https://i2y.github.io/connecpy/install.sh | PROTOC_GEN_CONNECPY_INSTALL=$HOME/.local/bin bash

# Install specific version
curl -sSL https://i2y.github.io/connecpy/install.sh | VERSION=v2.1.0 bash

📦 Install Python Package

After installing the protoc plugin, install the Python runtime:

# Using pip
pip install connecpy

# Using uv (recommended)
uv add connecpy

🎯 Generate Code

Using Buf (v2) - Recommended

Create a buf.gen.yaml file:

version: v2
plugins:
  - remote: buf.build/protocolbuffers/python
    out: gen
  - remote: buf.build/protocolbuffers/pyi
    out: gen
  - local: protoc-gen-connecpy
    out: gen

Then generate code:

buf generate

Using protoc directly

Generate Python code from your .proto files:

protoc --python_out=./ --pyi_out=./ --connecpy_out=./ ./your_service.proto

Using grpc_tools

python -m grpc_tools.protoc --python_out=. --pyi_out=. --connecpy_out=. -I . your_service.proto
